Something cool

I learned from the readings that semantic elements (for example, headings) are more about signifying data and doesn't really take on the role of design. As a tech writer, I think about visual layouts on pages a lot, which is more the job of CSS than HTML. This makes sense as an H1 looks large and in charge but can actually be designed to look any way you want with CSS, while still retaining the semantic value of an H1.

So much more to learn.
